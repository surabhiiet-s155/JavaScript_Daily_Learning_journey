# 📅 Day 12 - Form Validation

## 🚀 What I Learned
- Handling form submission
- Validating user input
- Showing error messages dynamically
- Preventing form submission if invalid

## 💻 Project
A form where:
- User enters name, email, password
- Errors are shown if input is invalid
- Form submits only when all inputs are correct

## 📂 Files Included
- index.html
- style.css
- script.js

## 🎯 Key Concepts
- onsubmit event
- Validation logic
- String checking
- Conditional statements

## ✨ Output
- Empty fields → Show errors
- Invalid email → Show message
- Short password → Show error
- Valid input → Form submits
