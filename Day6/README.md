# 📅 Day 06 - To-Do List App

## 🚀 What I Learned
- Creating elements dynamically
- Working with lists (ul, li)
- Handling user input
- Adding and removing elements
- Event handling

## 💻 Project
A simple To-Do app where:
- User can add tasks
- Tasks appear in a list
- Each task can be deleted

## 📂 Files Included
- index.html
- style.css
- script.js

## 🎯 Key Concepts
- document.createElement()
- appendChild()
- remove()
- Event handling (onclick)

## ✨ Output
- Add task → appears in list
- Click "X" → task deleted
