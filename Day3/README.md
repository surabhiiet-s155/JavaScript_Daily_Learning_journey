# Day 03 - Counter App

## 🚀 What I Learned
- Variables to store data
- Functions for actions
- Event handling (button clicks)
- Updating UI dynamically
- Code reusability using functions

## 💻 Project
A simple counter app where:
- User can increase the count
- Decrease the count
- Reset the count to zero

## 📂 Files Included
- index.html
- style.css
- script.js

## 🎯 Key Concepts
- Global variable (count)
- Functions for each action
- DOM manipulation using innerText

## ✨ Output
- Click "+" → Increase count
- Click "-" → Decrease count
- Click "Reset" → Set count to 0
