# 📅 Day 07 - Calculator App

## 🚀 What I Learned
- Handling multiple button inputs
- String manipulation
- Basic expression evaluation
- Error handling using try-catch

## 💻 Project
A simple calculator that:
- Takes input from buttons
- Performs basic operations (+, -, *, /)
- Shows result instantly

## 📂 Files Included
- index.html
- style.css
- script.js

## 🎯 Key Concepts
- appendValue()
- eval()
- slice()
- try-catch

## ✨ Output
- Click numbers/operators → display updates
- Click "=" → result shown
- Click "C" → clear display
- Click "DEL" → delete last input
